# 步骤 11：收尾清理

## 目标

在当前产品全链路完成后，只保留该产品无水印主图归档文件夹，清理其他全部过程中产生的文件，避免占用电脑内存或影响下一个产品流程。

## 动作脚本

- [cleanup.ts](src/autolist/cleanup.ts)

## 输入

- 当前运行目录
- 当前产品白底图
- 当前产品对应的标题表格
- 当前产品对应的 Word 文档
- 当前产品对应的 Dreamina 产物
- 当前产品对应的产品图片文件夹
- 当前产品对应的店铺目录残留
- 当前产品无水印主图归档路径

## 输出

- 无水印主图归档记录
- 已删除路径列表

## 操作说明

### 清理原则

```text
当一个产品从第一步开始到最后上架任务完成后，只允许保留没有添加水印的主图。无水印主图必须放入以本产品用户认知名命名的文件夹，并保存到：
/Users/mfrank/Desktop/FFC的文件夹/工作/001电商/2026AI主图

除这个归档文件夹外，当前产品的白底图、资质下载附件、Word 文档、标题表、运行目录、店铺产品文件夹、水印图、中间请求响应文件都必须清理。
```

## 执行要求

1. 必须先归档当前产品无水印主图，再删除运行目录。
2. 归档文件夹名称必须等于当前产品 `用户认知名`。
3. 医疗器械和保健食品应归档 20 张无水印主图；非处方药应归档 12 张无水印主图。
4. 必须清理当前产品白底图和资质下载附件。
5. 必须清理该产品对应的 Word、Dreamina 产物、标题表格、产品图片文件夹、店铺目录残留。
6. 不允许把当前产品残留留给下一个产品继续使用。

## 失败条件

- 白底图残留
- 资质下载附件残留
- 标题表格残留
- 产品图片文件夹残留
- 店铺目录中残留当前产品文件
- 无水印主图未归档

## 人工检查点

1. 检查 `/Users/mfrank/Desktop/FFC的文件夹/工作/001电商/2026AI主图/<用户认知名>/` 是否存在无水印主图。
2. 检查白底图目录和资质目录是否仍残留当前产品图片。
3. 检查店铺目录是否仍残留当前产品文件夹或图片。
4. 检查运行目录中是否还有当前产品的中间文件。
