# 步骤 11：收尾清理

## 目标

在当前产品白底图全链路完成后，清理该产品从第一步到最后一步产生的全部相关文件，避免任何残留影响下一个产品白底图的流程。

## 动作脚本

- [cleanup.ts](src/autolist/cleanup.ts)

## 输入

- 当前运行目录
- 当前产品白底图
- 当前产品对应的标题表格
- 当前产品对应的 Word 文档
- 当前产品对应的 Dreamina 产物
- 当前产品对应的产品图片文件夹
- 当前产品对应的店铺目录残留

## 输出

- 清理记录
- 已删除路径列表

## 操作说明

### 清理原则

```text
当一个产品白底图从第一步开始到最后上架任务完成后，一定要把这个产品白底图以及全流程产生的相关文件全部清除，防止任何残留影响下一个产品白底图的全链路流程。
```

## 执行要求

1. 必须清理当前产品白底图。
2. 必须清理该产品对应的 Word、Dreamina 产物、标题表格、产品图片文件夹、店铺目录残留。
3. 不允许把当前产品残留留给下一个产品继续使用。

## 失败条件

- 白底图残留
- 标题表格残留
- 产品图片文件夹残留
- 店铺目录中残留当前产品文件

## 人工检查点

1. 检查白底图目录是否仍残留当前产品图片。
2. 检查店铺目录是否仍残留当前产品文件夹或图片。
3. 检查运行目录中是否还有当前产品的中间文件。
