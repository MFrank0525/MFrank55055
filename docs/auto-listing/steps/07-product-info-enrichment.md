# 步骤 07：商品信息回填

## 目标

从商品信息表中抓取并回填每个产品图片文件夹内标题表格所需的商品信息。

## 动作脚本

- [metadata.ts](src/autolist/metadata.ts)

## 输入

- `./input/auto-listing/product-info.xlsx`
- 产品图片文件夹中的标题表格
- 可选映射文件：`product-info-key-map.json`

## 输出

- 已回填的标题表格

## 操作说明

### 必须回填的字段

```text
导购短标题
品牌
SPU信息
```

### 执行原则

```text
一定要按照表格里的信息执行，包括标题、导购短标题、品牌信息、SPU信息等，不要盲目去猜测，按照表格里面的信息执行即可。
```

## 执行要求

1. 必须严格按照表格里的信息执行。
2. 不允许程序自行猜测、替换、推导或自动选相近值。
3. `导购短标题`、`品牌`、`SPU信息` 任一为空都必须立即失败。
4. 可优先使用映射文件，不稳定时不允许回退到模糊匹配。

## 失败条件

- 匹配不到商品信息
- 回填字段有空值
- 程序试图用猜测值代替表格值

## 人工检查点

1. 打开任意一个产品文件夹里的标题表格，确认已写入导购短标题、品牌、SPU信息。
2. 核对回填值是否与商品信息表一致。
