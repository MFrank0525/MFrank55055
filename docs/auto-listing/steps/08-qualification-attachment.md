# 步骤 08：资质图片补充

## 目标

为每个产品图片文件夹补齐资质图片。

## 动作脚本

- [qualifications.ts](C:/Users/ffc/.openclaw/workspace/douyin-title-generator/src/autolist/qualifications.ts)

## 输入

- `C:\Users\ffc\Desktop\20260401产品资质图片`
- 产品图片文件夹列表
- 产品名称

## 输出

- 含资质图片的产品图片文件夹

## 操作说明

### 详情图规则

```text
产品图片文件夹里的资质图片，就是后续上架商品时详情页指定要上传的详情页图片。
```

## 执行要求

1. 以产品名称匹配资质图片。
2. 每个产品图片文件夹都必须执行此步骤。
3. 不允许只补一部分产品。
4. 产品图片文件夹里的资质图片，就是后续上架商品时详情页指定要上传的详情页图片。

## 失败条件

- 某个产品图片文件夹缺少资质图
- 资质图复制到错误产品文件夹

## 人工检查点

1. 检查每个产品文件夹里是否都有资质图。
2. 检查资质图是否明显与该产品相关。
