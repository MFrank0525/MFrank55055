# 步骤 08：资质图片补充

## 目标

为每个产品图片文件夹补齐资质图片。

## 动作脚本

- [qualifications.ts](src/autolist/qualifications.ts)

## 输入

- `./input/auto-listing/qualifications`
- 产品图片文件夹列表
- 产品名称

## 输出

- 含资质图片的产品图片文件夹

## 操作说明

### 详情图规则

```text
产品图片文件夹里的资质图片，就是后续上架商品时详情页指定要上传的详情页图片。
```

## 执行要求

1. 以产品名称匹配资质图片。
2. 每个产品图片文件夹都必须执行此步骤。
3. 不允许只补一部分产品。
4. 产品图片文件夹里的资质图片，就是后续上架商品时详情页指定要上传的详情页图片。
5. 资质图片文件名中的 `资质图片-01`、`资质图片-02`、`资质图片-03` 是飞书顺序，详情页上传必须按该顺序递增执行；哈希后缀只用于去重，不得参与排序。

## 失败条件

- 某个产品图片文件夹缺少资质图
- 资质图复制到错误产品文件夹

## 人工检查点

1. 检查每个产品文件夹里是否都有资质图。
2. 检查资质图是否明显与该产品相关。
