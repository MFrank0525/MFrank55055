# 步骤 09：店铺目录确认

## 目标

确认产品图片文件夹都位于正确店铺目录。

## 动作脚本

- [shop-distribution.ts](src/autolist/shop-distribution.ts)

## 输入

- 产品图片文件夹
- 店铺目录根路径
- 飞书产品类目

## 输出

- 就位的店铺目录结构

## 操作说明

### 店铺确认原则

```text
先以店铺编号为最高优先级确认目标店铺，再检查店铺名称是否一致。
```

## 执行要求

1. 已经在店铺目录内的产品文件夹不重复移动。
2. 未在店铺目录中的产品文件夹才执行分发。
3. 以店铺编号为最高优先级确认目标店铺。
4. 医疗器械和保健食品必须分发到 01-10 十个店铺。
5. 非处方药只允许分发到 01、02、03、04、05 五个店铺。

## 失败条件

- 产品文件夹进入错误店铺
- 同一产品文件夹被复制到多个店铺
- 非处方药进入 06、07、08、09、10 店铺

## 人工检查点

1. 抽查几个产品文件夹，确认所在店铺目录正确。
2. 检查是否存在重复产品文件夹。
