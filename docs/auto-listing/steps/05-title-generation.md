# 步骤 05：标题生成

## 目标

基于当前产品信息生成 20 条标题，并落成标题表格。

## 动作脚本

- [title-sheets.ts](C:/Users/ffc/.openclaw/workspace/douyin-title-generator/src/autolist/title-sheets.ts)

## 输入

- 当前产品卖点
- 当前白底图
- 固定标题豆包对话

## 输出

- 20 条标题
- 标题工作簿文件

## 操作说明

### 固定标题对话

```text
https://www.doubao.com/chat/38420067428736258
```

### 标题指令前缀

```text
请严格执行全套标题生成规范：
```

## 执行要求

1. 不新开标题对话，只复用固定 URL 对话。
2. 每次发送的标题指令必须以固定前缀开头。
3. 标题规则以当前步骤说明为准。
4. 输出格式固定为 `01 标题内容` 到 `20 标题内容`。
5. 只允许输出标题，不允许解释、备注、提示词。

## 失败条件

- 不能稳定提取到 20 条标题
- 输出里混入解释、说明、免责声明
- 标题表格未成功落地

## 人工检查点

1. 检查是否正好生成 20 条标题。
2. 检查输出是否只有标题正文。
3. 检查标题表格是否成功生成。
